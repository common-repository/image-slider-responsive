== Description ==

A WordPress Image slider based on FlexSlider to display image slider on post, page or theme. It's an easy, simple & responsive slider, which looks awesome in every single theme.

= Features =
* Fully responsive.
* Support Carousel Slider Options.
* Supported in all major browsers.
* Horizontal/vertical slide and fade animations.
* Hardware accelerated touch swipe support
* Navigation and pagination options
* Support image caption

Just create your slides from the Slider menu & paste the generated shortcode where you want to display this slide:

== Installation ==
* From your WordPress dashboard go to **Plugins > Add New**.
* Search for **Image Slider Responsive** in **Search Plugins** box.
* Find the WordPress Plugin named **Image Slider Responsive** by **Sayful Islam**.
* Click **Install Now** to install the **Image Slider Responsive** Plugin.
* The plugin will begin to download and install.
* Now just click **Activate** to activate the plugin.

If you still need help. visit [WordPress codex](https://codex.wordpress.org/Managing_Plugins#Installing_Plugins)
